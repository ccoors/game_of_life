# Game of Life

## Introduction
This is a simple Game of Life implementation in C++14.

## Features
* Square grid
* Standard 23/3 rule
* Multithreaded (up to 32 threads)
* Supports loading RLE files

### Planned features
* Non-square grid
* Other rules
* Support for other file formats
* Saving

## Build instructions
    mkdir build
    cd build
    cmake ..
    make
